## Why GitHub Issues?

- Single source of truth across all branches
- No merge conflicts in TODO tracking
- Better collaboration features
- Can link issues to PRs
- Track status without code changes

## Quick Commands

```bash
# List all open issues
gh issue list

# Create new issue
gh issue create

# View specific issue
gh issue view 25
```
